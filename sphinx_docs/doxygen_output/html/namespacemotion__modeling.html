gtsam::Point2 motion_modeling::calc_lateral_velocity_error ( double  ang_velocity,
double  yaw 
)

Calculates the lateral velocity error as a result of turning.

The IMU is offset from the center of the car. There's extra lateral velocity error due to the offset.

Returns
Returns a pair where the first element is the new_pose and the second element is the odometry.

Intuition: In the local frame, create a triangle T from the global bearing of the car

ang_velocity represents the magnitude of the angular velocity vector

  • To get the x component (global frame x), let the vertical side of the triangle be the radius
  • To get the y component (global frame y), let the horizontal side of the triangle be the radius
